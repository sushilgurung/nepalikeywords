How to use Nepali type KeyBoards Plugins

Initilize nepali keyboards

$('#keywords').nepaliKeyBoards();
Default Auto Complete is Off
$('#keywords').nepaliKeyBoards({ autocomplete:true });
Press Esc to change the language from nepali to english
